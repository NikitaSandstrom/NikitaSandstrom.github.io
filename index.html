The Web API Project

Description:

For this project, the task was to use one of 7 Api's given to us and to display 6 or more of the things the Api could display. With this Api I chose, it could display Spotify Albums from an artist (In this case, Scooter). This project focused heavily with AJAX for it's ability to grab data like this with aid from Jquery, and very minimal html and css.
